# 小红书违规词与红线库（全平台通用）

> 本文件只收**全平台通用红线**。各赛道的专属红线（如 AI 工具赛道的翻墙/镜像/破解教程、美妆赛道的功效宣称、母婴赛道的婴幼儿食品宣称等）见 `niche-<赛道>.md` 的「专属红线」部分。
> 核心原则：**不靠谐音/图片规避审核**。谐音规避是高风险玩法——平台识别能力一直在升级，今天管用的谐音明天就可能批量限流。本词库只列红线和**合规替代**，不教钻空子。

## 一、引流红线（最容易踩，重点）

这是主理人踩过坑的重灾区。一篇笔记结尾写「评论区留言或私信获取」直接被判违规。

### 绝对不能写的引流话术

| 违规表达 | 为什么危险 |
|---------|-----------|
| 私信我 / 私聊我 / 私信获取 | 直接引导私信，平台重点打击 |
| 评论区留言 / 评论扣 1 / 评论领 | 引导评论互动来发资料，典型引流 |
| 加微信 / 加群 / VX / 薇信 / 加 V | 站外引流，谐音也照样识别 |
| 留联系方式 / 留个 XXX | 变相引流 |
| 看主页 / 看简介 / 主页有 | 引导到简介区找联系方式 |
| 扫码 / 二维码 | 引流到站外 |
| 某宝 / 橙色软件（指代淘宝） | 谐音规避，同样违规 |

### 合规替代方案（不是换谐音，是换思路）

想要资料交付或建立联系，走合规路径，而不是在笔记里埋钩子：

| 想达成 | 合规做法 |
|---------|---------|
| 发教程资料/模板 | 用小红书「店铺」或「专栏」功能合规交付，或直接把干货写进正文 |
| 让粉丝联系你 | **被动等粉丝私信你**（合规），而不是你写「私信我」 |
| 引导关注 | 写「关注我，持续分享 XX」即可，**不带获取动作** |
| 建立信任 | 靠内容质量和人设，不靠引流钩子 |

> 关键认知：**钩子（评论领、私信领）常常是违规重灾区**。把价值直接放进正文，让笔记本身值得收藏，比设钩子更安全也更有效。

## 二、广告法绝对化用语（全平台通用）

这些词触发广告法，笔记会被限流甚至下架：

- **极端词**：最、第一、顶级、极品、极致、万能、永久、独家、首创、唯一
- **权威词**：国家级、最高级、最佳、第一品牌、销量冠军、全网首发
- **效果承诺**：包教包会、稳赚、保本、零风险、100% 有效

**替代思路**：用「真实体验 + 具体场景」替代绝对化承诺。说「我用下来感受是 X」比说「全网最好」既安全又有说服力。

## 三、医疗 / 金融类（需资质）

无资质不碰，碰了直接违规：

- **医疗**：治疗、治愈、疗效、药方、偏方、医美、整形（需医师资质）
- **化妆品**：美白（需特证）、祛斑、祛痘印、抗衰老（需依据）
- **金融**：稳赚、保本、高收益、零风险、内幕、荐股（需金融资质）

## 四、笔记被限流的常见原因（按概率排序）

笔记数据异常时，按这个顺序自查：

1. **含引流话术或敏感词**——最常见，先全文搜一遍上文红线词
2. **非原创/搬运**——直接搬别人内容，平台识别
3. **短时间内大量发布**——被判定营销号（一天发 5+ 篇危险）
4. **频繁修改笔记（超过 3 次）**——每次改都重新审核，改多了降权
5. **评论区出现违规引流**——自己或粉丝在评论里留联系方式
6. **账号有刷量行为**——买赞买粉，平台一查一个准
7. **新号高频发同质内容**——还没建立信任就刷屏

## 五、被限流后的处理

1. **先别慌改**：确认是不是真的违规。有时只是流量池没起来，不是限流。
2. **对照本词库自查**：把笔记里疑似敏感的表达改掉（改成合规表达，**不是改成谐音**）。
3. **改稿不超过 1-2 次**：频繁改反而持续降权。
4. **实在不行就删了重发**：换标题、换封面、调整正文，当新笔记发，比硬救老笔记有效。
5. **复盘原因记下来**：踩过的坑写进经验，下次生成内容前先避开。
