# 赛道参考包：穿搭时尚

## 一、赛道画像与数据

小红书第二大种草赛道，与服饰电商深度绑定，变现靠带货，门槛低易上手但极度内卷。

| 细分方向 | 竞争度 | 流量趋势 | 变现能力 | 内容门槛 |
|---------|--------|---------|---------|---------|
| 日常 OOTD | 极高 | 平稳 | 高 | 低 |
| 身材穿搭（小个子/微胖/梨形） | 高 | 上升 | 高 | 中 |
| 职场通勤 | 高 | 平稳 | 高 | 低 |
| 一衣多穿/胶囊衣橱 | 中 | 上升 | 中 | 中 |
| 汉服/亚文化/Y2K | 中 | 上升 | 中 | 中 |
| 男士穿搭 | 低 | 上升 | 中 | 低（蓝海） |

**受众画像**：18-35 岁女性为主，看重性价比、场景适配和身材修饰。决策快，容易被「同款」种草。

**机会**：男士穿搭、特定身材（小个子/梨形/苹果型）、胶囊衣橱是差异化方向；「一衣多穿」涨收藏强。

**风险**：同款抄袭、假货问题、身材焦虑内容敏感、价格误导。

## 五、爆款案例拆解

### 案例 1：身材穿搭公式（精准涨粉）
- **标题**：「梨形身材夏天这么穿显瘦」（12 字）
- **公式**：身材标签 + 场景 + 效果
- **结构**：身材痛点 → 公式（上装+下装+鞋）→ 每套配图 → 适用人群 → 互动
- **可复用点**：精准切身材人群，评论区全是「同款身材求链接」，涨粉快

### 案例 2：一衣多穿（涨收藏）
- **标题**：「一件衬衫的 5 种穿法」（11 字）
- **公式**：单品 + 数字 + 价值
- **结构**：单品展示 → 5 种搭配（每套配图 + 场景）→ 适用季节 → 收藏
- **可复用点**：一衣多穿收藏率极高，凸显搭配功力

### 案例 3：平价通勤（易传播）
- **标题**：「通勤这么穿，被夸像博主」（12 字）
- **公式**：场景 + 效果 + 社交背书
- **结构**：通勤痛点 → 3 套搭配 → 单品价格 → 适用人群 → 互动
- **可复用点**：通勤场景受众广，平价 + 好看易传播

### 案例 4：避雷对比（涨互动立信任）
- **标题**：「微胖别这么穿，显胖 10 斤」（13 字）
- **公式**：人群 + 否定 + 痛点
- **结构**：错误穿法（配图）→ 为什么显胖 → 正确穿法 → 总结 → 互动
- **可复用点**：对比类天然带话题，立「懂行」人设

### 案例 5：季节胶囊衣橱（收藏天花板）
- **标题**：「秋冬胶囊衣橱，12 件搞定」（13 字）
- **公式**：季节 + 概念 + 数字
- **结构**：胶囊衣橱概念 → 12 件清单（基础+亮点）→ 互相搭配 → 收藏
- **可复用点**：合集类收藏天花板，体系化内容带关注

### 通用爆款要素（本赛道）
1. 标题 ≤20 字，必有身材标签/场景/数字/价格之一
2. 封面：全身穿搭照 + 身高体重标签 或 前后对比
3. 真实上身效果，配身高体重让用户代入
4. 价格透明，平价单品更易传播
5. 结尾不设引流钩子
6. 标签：核心词（#穿搭 #OOTD）+ 长尾（#梨形身材 #小个子穿搭）+ 热门（#秋冬穿搭）
