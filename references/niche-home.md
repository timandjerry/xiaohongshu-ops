# 赛道参考包：家居生活

> 本文件是「家居生活」赛道的参考包。通用运营框架见 SKILL.md，通用红线见 banned-words.md，本文件只放赛道专属内容。

## 一、赛道画像与数据

近两年上升最快的赛道之一，与家居电商深度绑定，变现靠带货和品牌投放，受众消费力强。

| 细分方向 | 竞争度 | 流量趋势 | 变现能力 | 内容门槛 |
|---------|--------|---------|---------|---------|
| 出租屋/租房改造 | 中 | 上升 | 中 | 低 |
| 小户型收纳整理 | 中 | 上升 | 中高 | 中 |
| 软装搭配/灵感 | 中高 | 上升 | 高 | 中 |
| 家电/清洁好物 | 中 | 上升 | 高 | 低 |
| 装修避坑/干货 | 中 | 平稳 | 中高 | 中高 |
| 极简/侘寂/奶油风 | 中 | 上升 | 中 | 中 |

**受众画像**：25-35 岁，租房或新房装修人群为主，一二线城市居多。看重颜值、性价比和实用性，决策周期较长但客单价高。

**机会**：收纳整理、清洁家电是涨粉带货双强的方向；小众风格（侘寂/中古）差异化；「低成本改造」易传播。

**风险**：装修质量夸大、环保/甲醛虚假宣称、价格误导、收纳神器踩雷。

**为什么这么分**：改造前后对比是家居赛道涨粉核武器；避雷类立「懂行」人设。

## 五、爆款案例拆解

### 案例 1：出租屋改造（涨粉核武器）
- **标题**：「500 块改造出租屋，变温馨」（14 字）
- **公式**：预算 + 动作 + 反差
- **结构**：改造前（破旧）→ 改造思路 → 分区改造（配图+花费）→ 改造后 → 复盘
- **可复用点**：before/after 视觉冲击极强，低预算易传播，涨粉快

### 案例 2：收纳技巧（涨收藏）
- **标题**：「厨房收纳，台面瞬间清爽」（13 字）
- **公式**：场景 + 动作 + 效果
- **结构**：痛点（乱）→ 分区收纳方法 → 工具推荐 → 效果对比 → 收藏
- **可复用点**：收纳类收藏率高，强种草

### 案例 3：好物开箱（涨互动带货）
- **标题**：「这些收纳神器真香」（11 字）
- **公式**：品类 + 评价
- **结构**：钩子（提升幸福感）→ 逐个产品（用途+价格+体验）→ 避雷款 → 互动
- **可复用点**：好物合集带货转化好

### 案例 4：避雷踩坑（立信任）
- **标题**：「这些网红家居别买，踩雷了」（14 字）
- **公式**：否定 + 痛点
- **结构**：踩雷产品 → 为什么难用 → 替代方案 → 总结 → 互动
- **可复用点**：避雷类立「懂行」人设，互动高

### 案例 5：风格灵感（涨收藏）
- **标题**：「奶油风客厅，直接抄作业」（13 字）
- **公式**：风格 + 场景 + 指令
- **结构**：风格定义 → 色彩/材质搭配 → 单品清单 → 避免踩雷 → 收藏
- **可复用点**：风格类收藏天花板，「抄作业」降低决策成本

### 通用爆款要素（本赛道）
1. 标题 ≤20 字，必有预算/场景/风格/否定之一
2. 封面：before/after 对比图 或 风格实景大图
3. 视觉是第一生产力，家居颜值即正义
4. 价格、尺寸透明，让人能直接抄
5. 结尾不设引流钩子
6. 标签：核心词（#家居 #收纳）+ 长尾（#出租屋改造 #奶油风）+ 风格词
