## 一、赛道画像与数据

小红书第一大种草赛道，品牌投放最密集，变现天花板最高，但竞争也最卷。

| 细分方向 | 竞争度 | 流量趋势 | 变现能力 | 内容门槛 |
|---------|--------|---------|---------|---------|
| 护肤（成分/功效） | 极高 | 平稳 | 极高 | 中高（要懂成分） |
| 彩妆（教程/测评） | 极高 | 平稳 | 高 | 中 |
| 香水/香氛 | 中高 | 上升 | 中 | 低 |
| 个护（身体/口腔/头发） | 中 | 上升 | 中 | 低 |
| 男士理容 | 低 | 上升 | 中 | 低（蓝海） |

**受众画像**：18-35 岁女性为主，一二线城市居多。决策高度受 KOL/KOC 影响，愿为「种草」买单，但对成分和真实感越来越挑剔，反感硬广和过度修图。

**机会**：男士理容、个护是相对蓝海；成分党内容门槛高、信任强；「真实肤质记录」「空瓶记」类真实内容持续吃香。

**风险**：功效宣称红线多（美白/祛斑/抗衰需特证或依据）、医美内容需资质、虚假对比图易翻车。

## 五、爆款案例拆解

> 标题普遍 **≤20 字**、emoji 克制、口语化。

### 案例 1：妆容教程类（涨收藏）
- **标题**：「新手 3 步画出妈生眉」（11 字）
- **公式**：人群 + 步骤 + 效果承诺
- **结构**：痛点（手残画不好）→ 工具准备 → 分步演示（配图）→ 成品对比 → 收藏引导
- **可复用点**：教程类每步配图、结果可视化，让人相信「我也能学会」

### 案例 2：粉底测评类（涨互动）
- **标题**：「5 款平价粉底液，谁是真爱」（13 字）
- **公式**：数字 + 品类 + 悬念结论
- **结构**：测评维度（持妆/遮瑕/色号）→ 逐款打分（表格）→ 分肤质推荐 → 互动
- **可复用点**：分肤质/分人群给结论，显得中立，评论区会吵起来

### 案例 3：护肤蜕变类（立人设涨粉）
- **标题**：「敏肌养厚屏障，用了 3 个月」（14 字）
- **公式**：肤质痛点 + 时间 + 真实经历
- **结构**：起因（烂脸）→ 用了什么 + 怎么用 → 过程记录 → 现在状态 → 复盘
- **可复用点**：美妆赛道涨粉核武器，靠「真实过程」打动人

### 案例 4：平价好物类（易传播）
- **标题**：「10 块钱好物，被闺蜜追问」（12 字）
- **公式**：价格反差 + 社交背书
- **结构**：钩子（便宜好用）→ 产品展示 → 使用场景 → 价格揭秘 → 互动
- **可复用点**：价格反差制造话题，真实感强

### 案例 5：成分科普类（涨收藏立专业）
- **标题**：「烟酰胺到底怎么用才有效」（12 字）
- **公式**：成分名 + 痛点疑问
- **结构**：成分是什么 → 解决什么 → 怎么用（浓度/搭配禁忌）→ 避坑 → 收藏
- **可复用点**：成分党内容门槛高、收藏率高，立专业人设

### 通用爆款要素（本赛道）
1. 标题 ≤20 字，必有数字/肤质痛点/价格反差/悬念之一
2. 封面：妆容特写或前后对比，大字 + 肤质标签
3. 真实感第一：不过度修图、不硬广，第一人称体验
4. 干货可复现：教程能照做、测评有真实维度
5. 结尾不设引流钩子，「关注持续分享」即可
6. 标签：核心词（#美妆 #护肤）+ 长尾（#油皮护肤 #敏肌）+ 热门（#年度爱用）
